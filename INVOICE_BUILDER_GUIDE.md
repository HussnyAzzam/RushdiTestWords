# Israeli Invoice Builder - Implementation Guide

## Overview
The Israeli Invoice Builder is a fully client-side tool for creating professional business documents in Hebrew, designed specifically for Israeli businesses.

## Configuration Structure

The app uses a central `invoiceSettings` object that includes:

```js
- locale: "he-IL"
- direction: "rtl"
- currency: { code: "ILS", symbol: "₪" }
- tax: { vatName: "מע״מ", defaultVatRate: 18 }
- businessTypes: { exempt, licensed, company }
- documentTypes: { "חשבונית מס", "קבלה", ... }
- unitTypes: { unit, hours, service, project, product, other }
- paymentMethods: { transfer, check, cash, creditCard }
```

2. **Enter business details** → Business Setup Form
3. **Create new document** → Document Creator
4. **Select document type** → Choose from 7 document types
5. **Add line items** → Dynamic line items editor
6. **Add customer details** → Customer form
7. **Live preview** → Real-time preview
8. **Customize** → Colors, fonts, fields visibility
9. **Export** → PDF or PNG

## Calculation Logic

### Line Item Calculation
```
Gross = Quantity × Unit Price
Discount = (Gross × Discount%) OR Fixed Discount
Row Total = Gross - Discount
```

### Invoice Summary
```
Subtotal = Sum of all row totals
Global Discount = Applied to subtotal
Subtotal after Discount = Subtotal - Global Discount
VAT Amount = Subtotal after Discount × VAT Rate / 100
Total = Subtotal after Discount + VAT Amount
```

### Business Type Rules
- **עוסק פטור**: VAT disabled (0%)
- **עוסק מורשה**: VAT enabled (default 18%)
- **חברה בע״מ**: VAT enabled (default 18%) + Company number field

## Data Persistence

Using Browser LocalStorage:
- Last used business details
- Last used document numbers
- User preferences
- Draft documents

## Important Notes

1. **No Backend**: All processing happens in the browser
2. **Privacy**: All data stays on user's device
3. **No API Calls**: Fully independent
4. **No Authentication**: Direct access
5. **Numbering**: User is responsible for legal numbering continuity
6. **Allocation Number**: Not generated by app, manually entered by user
